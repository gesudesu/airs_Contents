第5章　反応　　STEP5：条件分岐②

問題のコード

プログラムの実行画面

問題文

  • プログラムを実行結果の図になるようにしたい。
  • (1) iの条件式に当てはまるものを解答欄から選択してください。
  • (2) jの条件式に当てはまるものを解答欄から選択してください。

ヒント
どこに条件式が適用されているのかに注目してみよう!
実行結果を見ると、条件式が真の場合に赤く塗られていることがわかる
赤が適用されているのは、iとj共に値が3,4,5,6の時である。
iとjの初期値が0であることを忘れずに！

※　xが0以上、9未満という条件式を書くときには、
0 <= x < 9
では、動作してくれない！
論理演算子を用いて、記述しなければならない
0 <= x && x < 9

解答欄

ア： 2 < i && i < 7 イ： 2 < i < 7 ウ： i < 3 && i > 7 エ： 2 < i || i < 7 ア： 2 < j && j < 7 イ： 2 < j < 7 ウ： j < 3 && j > 7 エ： 2 < j || j < 7
