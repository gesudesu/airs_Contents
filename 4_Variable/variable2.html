問題文

  • 左図は、赤い円を4つ生成するプログラムです。
  • (1) int型の変数step, x, y, lを宣言しているもの選択してください。
  • (2) 実行画面を参考にlの値を示すものとして、正しいプログラムを選択してください。

ヒント

(2)　実行結果をよくみて考えてみよう！
lの値　：　円の直径
円はどんどん大きくなっているので、
l = l + 変数;　と表す。
上の変数に当てはめるのは何か、円がどのくらい大きくなっているのかに注目して考えてみよう。

解答欄

ア： int step, x, y, l; イ： int step x y l; ウ： float step, x, y, l; ア： l = l + x;　　　 イ： l = l + step; ウ： l = l + y;　　　 エ： l = l - step;
