解説

  • (1) for文の処理を書く問題です。変数iを用いて、円を3つ生成してください。
    • 正解：i = 0;　i < 3;　i++　/　i =0; i <= 2; i++
    for文は、繰り返しの処理を実現します。
    for文を用いることにより、STEP1のコードを短縮して書き換えることができます。
    for文の処理は以下のように記述します。
    for( 初期処理; 繰り返し条件; 繰り返し時の処理{
    繰り返す命令文
    }
    この条件に、円を3つ繰り返す処理を当てはめると、
    iの値が0(初期処理)から、　　i = 0;
    　　3未満(2以下）になるまで(繰り返し条件)　　i < 3;　or　i <= 2
    　　　　　増やし続ける(繰り返し時の処理)　　i++
    よって、i = 0; i < 3; i++　/　i =0;i<=2;i++が正解となる。
