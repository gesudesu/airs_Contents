 void setup(){
size(400,400); //ウインドウの大きさの指定

void draw(){
int timer = millis(); //変数timerの宣言

if(timer < 3000){ //3秒経過した時に
fill(255,0,0); //色の指定
}else if{timer < 6000){ //6秒経過した時に
fill(0, 255, 0); //色の指定
}else{
fill(0, 0, 255);
}

(1)　millis()の解説

関数millis()を用いることで、プログラムをスタートしてから経過時間をカウントして値を返すことができます。
ここでは、millis()関数を用いた変数timerを宣言しています。

(2)　timer < 3000の解説

時間(timer)が3000msまでの間、画面内の円の色が赤色に変更されるコードになっております。
1
 fill(255,0,0);






(3)　timer < 6000の解説

else if{}で先ほどの条件とは違う条件を適用します。
次の条件式は、時間(timer)が6000msまでの間、画面内の円の色が緑色に変更されるコードになっております。
1
 fill(0,255,0);






最後のelseでは、timer<3000でも、timer<6000でもない時ということを条件として表しています。
