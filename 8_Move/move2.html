問題のコード

プログラムの実行画面

問題文

  • 左図は、円を左から右へ移動させるプログラムです。
    円が画面の端に来た時には、跳ね返って左に戻るようにします。
  • (1)「画面の右端に来たら」という条件式に当てはまるものを選択してください。
  • (2)　条件式が真の時、「円を左に戻す」という処理として当てはまるものを選択してください。

ヒント
(1)　STEP1の問題を少し書き換えて、右端に行くと跳ね返るようにした問題です
画面の右端は、widthと表す
右端に円が接触するのは、円の中心の座標 + 円の半径がwidthを超えたとき
これに数値や変数を当てはめて、条件式を完成させてみよう！

(2)　円を跳ね返りで右から左へ戻るようにするときは、移動に関係している変数の符号を変えてあげればよい
移動に関係している変数は、speed
通常時　x += speed　により、xの値が徐々に増えていくようになっているが、
条件式が真の時に、このspeedを-speedに変更することにより、xの値が徐々に減っていき、右から左へ戻る動きを再現できる

解答欄

ア： x + radius/2 > width イ： x < radius
ウ： x + radius < width　 エ： x < width ア： x = -speed;　　　　 イ： speed = -speed;
ウ： x -= speed;　　　　 エ： speed = -x;
